# mouse_tracker

1) В файлах .log лежит консольынй вывод программ
2) cropped_mask(temp) - хранят маски и шаблоны курсора найденные на указанном кадре
3) edges - хранит изображение границ кадра
4) squared_cursor - хранит кадры на которых выделены найденные курсоры
5) test_frames - хранит обрабатываемые кадры + промежутиочные этапы их обработки
